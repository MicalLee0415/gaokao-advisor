# 报告视觉规范

国家级智库/顶级咨询公司风格。核心原则："内容为王，留白为骨"。

## 配色方案

### 主色

| 用途 | 色号 | 说明 |
|---|---|---|
| 页面底色 | #FFFFFF | 纯白 |
| 表头/一级标题/封面主色 | #1E3A5F | 深海蓝——权威、冷静、可信 |
| 风险/预警/负向结论 | #6E2C2C | 勃艮第红——比大红沉稳 |

| 冲档策略表头 | #2874A6 | 亮蓝——比深海蓝浅，提高对比度 |

## 排版

### 页边距

```css
@page { size: A4; margin: 22mm 13mm 22mm 13mm; }
```

### 行距

- 正文：1.6
- 表格：1.5-1.6
- 标题：1.3

### 表格规范

| 元素 | 规范 |
|---|---|
| 表头背景 | #1E3A5F（深海蓝） |
| 表头文字 | 白色加粗 |
| 行交替 | 奇数行#FFFFFF / 偶数行#F2F2F2 |
| 边框 | 1px solid #AAAAAA（细线浅灰） |
| 单元格padding | 7-8px |

### 强调色占比

- 白色（底色）：~85%
- 深灰（正文）：~10%
- 深海蓝（标题/表头）：~3%
- 勃艮第红/墨绿（强调）：~2%

**拒绝**：荧光色、高饱和红黄蓝、渐变色、装饰性元素。

## 封面规范

- 背景：纯白#FFFFFF
- 标题位置：中央偏上
- 标题色：深海蓝#1E3A5F
- 分隔：细横线（1px深海蓝）
- 副标题/日期：浅灰#888888
- 封底右下角："内部资料·请勿外传"（勃艮第红8pt）
- **不放置**任何图片、装饰图案、背景纹理
