# 西藏高考政策模板

## 基本信息

| 项目 | 值 |
|---|---|
| 高考模式 | 传统高考（文理分科） |
| 志愿数量 | 10个 |
| 志愿类型 | 院校+专业（1志愿=1院校+1专业/专业组） |
| 投档规则 | 平行志愿（分数优先，遵循志愿） |
| 调剂范围 | 同院校内专业调剂 |
| 批次设置 | 提前批 / 本科一批 / 本科二批 / 专科批 |

## 选科要求

- **文理分科**：文科（语数外+政史地）/ 理科（语数外+物化生）
- **志愿单位**：院校+专业，每校可填6个专业

## 专项计划

| 类型 | 条件 | 录取优惠 |
|---|---|---|
| 国家专项 | 指定贫困县户籍+3年学籍 | 单独批次，降分录取 |
| 地方专项 | 本省农村户籍+3年县域学籍 | 本科批内单独代码，降5-15分 |
| 高校专项 | 农村户籍+3年县域学籍+高校审核 | 提前申请，降分录取 |
| 免费医学定向 | 签协议去基层服务6年 | 免学费+有编制 |

## 冲稳保策略（西藏10志愿）

```
冲档：1-2个（位次×0.90~0.97）
稳档：5-7个（位次×1.00~1.25）
保档：2-3个（位次×1.25~1.70）
最后2个：位次必须>考生位次×1.5
```

## 关键提醒

1. 10个志愿**全部勾选"专业服从调剂"**
2. 调剂只在**同专业组内**（或同院校内），不会跨组
3. 平行志愿只有**一轮投档**，退档=直接掉到征集
4. 地方专项需提前确认资格（3年本省县域学籍+农村户籍）

## 省份特点

西藏仍实行文理分科传统高考。各批次可填10个院校志愿，每校6个专业。满分750分。
